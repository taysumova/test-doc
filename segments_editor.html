# Редактор сегментов

# Общее описание

Редактор сегментов - это гибкий механизм для формирования и расчета количества клиентов в режиме online.

TIP

Для использования в акции можно выбрать несколько различных сегментов.

WARNING

Сегментация производится на основе данных хранилища, данные в хранилище обновляются из системы Manzana Loyalty 1 раз в 10 минут.

# Методика расчета RFM

  • Контакты делятся по возрастным группам с границами [a, b). Если Возраст ребенка неизвестен, или >= 12, то контакт попадает в возрастную группу равным 2 года;

  • Расчёт R: Количество дней с последней покупки до даты расчёта RFM;

  • Расчёт F: Если количество визитов <= 1, то это наихудшая частота посещения и F присваивается сегмент 1. Если количество визитов > 1, то расчёт F идёт таким образом: (разница между первой и последней датой покупки) / (количество визитов – 1);

  • Расчёт M: Сумма покупки со скидкой / Количество визитов;

  • Учитываются покупки только по всем картам на момент расчёта RFM. Из расчёта исключаются позиции, попавшие в категории “РЕКЛАМНЫЕ МАТЕРИАЛЫ, ДИСКОНТНЫЕ КАРТЫ,СЕРТИФИКАТЫ” и “ СЫРЬЕ И МАТЕРИАЛЫ”;

  • Расчётный период 1 год;

  • Контактам, которые не попали в расчёт RFM проставляются значения 0-0-0. После этого отбрасываются контакты, которые отсутствуют в данный момент времени в CRM.

# Работа с редактором сегментов

  1. Для добавления нового сегмента нажмите плюс. Будет добавлен новый сегмент.
  2. Для добавления фильтра в сегмент выберите нужный - откроется окно конструктора запросов. Фильтры можно добавлять как простым нажатием, так и с помощью Drag & Drop.
  3. Фильтры можно группировать. Добавить новый фильтр в группу можно только с помощью Drag & Drop.
  4. После настройки сегмент можно выбрать с помощью кнопки "Использовать в акции".
  5. Для возврата в конструктор акции нажмите кнопку "В акцию"

WARNING

Каждый дочерний сегмент сохраняет фильтры родителя. Т.е. при добавлении дочернего сегмента фильтрация производится только по аудитории родительского сегмента.
