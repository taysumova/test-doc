# Ограничения в текущей версии

  1. Для проверки строки на наличие значения необходимо использовать оператор «Не пустое»
  2. При использовании оператора «между» в выборку включаются обе границы настраиваемого фильтра. Например, возраст ребенка: от 2 до 4. Значит будут использоваться значения 2, 3 и 4
  3. В фильтре «Мобильный телефон» для проверки формата номера, необходимо использовать следующую настройку – Мобильный телефон: «начинается с» +79
  4. Пересчет RFM производится один раз в месяц 31 числа.
  5. Расчет RFM производится на основе данных из DWH.
  6. Данные в системе Campaign обновляются 1 раз в 10 минут непосредственно из системы Manzana Loyalty. Из-за этого может быть разница между значениями фильтров в Campaign и выборками в кубах.

WARNING

Удалять можно только акции в статусе "Черновик"
