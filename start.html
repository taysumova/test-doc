# Работа в системе

# Что такое Manzana Campaign

Manzana Campaign – это инструмент автоматизации маркетингового взаимодействия, позволяющий вести персональную работу с клиентами, реагировать на действия пользователя в рамках маркетинговой акции, настраивать триггерные коммуникации.

- внешний вид ноды в конструкторе акций.

# Основные элементы системы

  • Шапка сайта - состоит из следующих элементов.
    • "Бутерброд и логотип" - позволяет вернуться на главную страницу и скрыть меню навигации.
    • "Хлебные крошки" - дополнительная навигация по сайту. Позволяет отслеживать путь перемещения по разделам.
    • "Пользователь" - информация о пользователе и возможность выйти из профиля.
  • Меню навигации - перемещение по разделам системы

# Алгоритм настройки акции

  1. Создавать акцию можно как из шаблона, так и непосредственно из списка акций.
  2. Если акция создается из шаблона, то необходимо настроить шаблон акции и нажать кнопку "Создать акцию".
  3. Акции и шаблоны акций можно копировать и удалять.
  4. Для настройки структуры акции перенесите необходимые элементы в "Конструктор акции".
  5. Соедините шаги для организации последовательности акции.
  6. Соединения и шаги можно удалять.
  7. Так же при нажатии на шаге правой кнопкой мыши можно вызвать контекстное меню. Оно позволяет удалить шаг, скопировать его, удалить все связи из шага и добавить примечание.
  8. Так же в акции можно выбрать несколько различных сегментов для дальнейшего взаимодействия с ними. Делается это в Редакторе сегментов
  9. В шагах рассылки СМС, Email и Viber для организации рассылки необходимо выбрать шаблон и, при необходимости, скорректировать его. Шаблоны создаются в разделе Шаблоны
  10. Если все шаги в акции настроены и валидны, то акцию можно активировать.
  11. Сразу после активации акции для шагов "Завершение", "Процентный разделитель", "Проверка условия", "СМС-сообщение", "Viber-сообщение", "Email-сообщение" и "Триггер коммуникации", в которых указана опция "Сохранить и выгрузить список", производится формирование списков участников и выгрузка их в систему ML с названием, которое указано в поле название списка.

# Статусы акции

В системе доступны следующие статусы акций:

  1. Черновик - акцию можно удалять и настраивать неограниченное количество времени.
  2. Запущена (активная) - акция выполняется и в ней демонстрируется реальное количество участников на каждом шаге.
  3. Остановлена - акция остановлена принудительно пользователем. Не подлежит повторной активации.
  4. Завершена - акция остановлена по завершении срока действия, указанного в шаге "Старт". Не подлежит повторной активации.

# Основные функциональные отличия от старой версии

  1. Из портрета участника может быть несколько выходов.
  2. Шаг "А/Б тестирование" заменен на шаг "Процентный разделитель".
  3. Тестирование коммуникации выполняется непосредственно в акции - в шаге SMS, Viber или Email, на вкладке "Тестирование".
